# Runtime skill packs

Runtime-specific skills live as **peer directories** next to universal skills under **`.apm/skills/`** (for example **`ecs-observability-troubleshooting/`**, **`eks-observability-stack/`**), each with its own **`SKILL.md`**.

Customer forks depend on this repo via **APM** (see root **`apm.yml`** and **kuberly-stack** **`docs/agent-packaging/`**).
